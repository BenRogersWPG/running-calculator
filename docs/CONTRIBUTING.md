# How to Contribute

I definitely welcome feature requests and enhancement contributions!
Here are some guidelines and information about how to participate.

## Getting Started

### Feature Requests 💡

If you want to submit a feature request or wish to report a bug, please see
[Issues](https://github.com/BenRogersWPG/running-calculator/issues).

### Pull Requests ⬇️

Please see the [Pull Requests](https://github.com/BenRogersWPG/running-calculator/pulls) section for more information about initiating a PR.

### Follow Code Updates 👨‍💻

If you are interested in following along with the code updates, please see the [GitHub repository](https://github.com/BenRogersWPG/running-calculator) where you can see the latest updates, fork and star the repository.

### Support Me 💕

If you have been enjoying this repository and wish to show your thanks, feel free to [sponsor me](https://github.com/sponsors/BenRogersWPG) - *there's some benefits for you too!*.

*Thanks so much!*
